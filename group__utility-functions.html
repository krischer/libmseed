Parameters
[in]sampletypeLibrary sample type code:
  • 'a' - Text/ASCII data type
  • 'i' - 32-bit integer data type
  • 'f' - 32-bit float data type
  • 'd' - 64-bit float (double) data type
Returns
The sample size based on type code or 0 for unknown.
